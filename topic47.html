📘 Берілген санның жұп немесе тақ екенін анықтайтын бағдарлама құру

Берілген санның жұп немесе тақ екенін анықтайтын бағдарлама — бағдарламалауда ең қарапайым шартты операторлар мен арифметикалық операцияларды қолдануға арналған классикалық есептердің бірі. Бұл есептің негізінде санның 2-ге бөлінгіштігін тексеру жатыр. Python тілінде толық бағдарлама: # Пайдаланушыдан сан енгізу n = int(input("Санды енгізіңіз: ")) # Санның жұп немесе тақ екенін тексеру if n % 2 == 0: print("Енгізілген сан жұп сан") else: print("Енгізілген сан тақ сан") Бағдарламаның жұмыс істеу принципі: Бұл бағдарламада негізгі логика modulo (%) операциясына негізделген. n % 2 — санды 2-ге бөлгендегі қалдықты табады. Егер n % 2 == 0 болса → сан жұп Егер n % 2 != 0 болса → сан тақ Мысалдар: 1-мысал: Санды енгізіңіз: 10 Енгізілген сан жұп сан 2-мысал: Санды енгізіңіз: 15 Енгізілген сан тақ сан Қосымша нұсқа (тернарлық оператор арқылы): n = int(input("Санды енгізіңіз: ")) print("Жұп сан" if n % 2 == 0 else "Тақ сан") Қорытынды: Бұл бағдарлама шартты тексеру (if-else) және арифметикалық операцияны қолдану арқылы жазылады. Ол алгоритмдік ойлауды дамытуға және бағдарламалаудың негізгі принциптерін түсінуге көмектеседі.
