📘 Локальды және глобальды айнымалылар

**Локальды және глобальды айнымалылар** — Data Structures және программалау негіздерінде кең қолданылатын маңызды ұғымдардың бірі. Олар айнымалылардың **көріну аймағын (scope)** және **өмір сүру уақытын** анықтайды. Яғни, программада айнымалы қай жерде қолжетімді екенін және қанша уақыт сақталатынын түсіндіреді. Бағдарламада айнымалылар деректерді сақтау үшін қолданылады, бірақ олардың барлығы бірдей деңгейде жұмыс істемейді. Кейбір айнымалылар тек белгілі бір функция ішінде ғана қолжетімді болса, кейбіреулері бүкіл программа бойында қолданылады. Осыған байланысты олар локальды және глобальды болып бөлінеді. --- ## 🔵 Глобальды айнымалылар (Global variables) **Глобальды айнымалы** — функциялардан тыс, яғни програманың негізгі бөлігінде жарияланатын және барлық функциялардан қолжетімді айнымалы. ### Ерекшеліктері: * Програманың кез келген жерінен қолдануға болады * Бір рет жарияланады және бүкіл программа бойы сақталады * Бірнеше функция ортақ дерек ретінде пайдалана алады * Үлкен программаларда ортақ мәндерді сақтау үшін қолданылады ### Мысал: ```python id="global_var_full" x = 100 # глобальды айнымалы def func1(): print("func1:", x) def func2(): print("func2:", x) func1() func2() print("Негізгі программа:", x) ``` Бұл жерде `x` айнымалысы барлық функцияда қолжетімді. --- ## 🟢 Локальды айнымалылар (Local variables) **Локальды айнымалы** — тек белгілі бір функцияның ішінде жарияланатын және тек сол функция ішінде ғана қолданылатын айнымалы. ### Ерекшеліктері: * Тек функция ішінде ғана көрінеді * Функция аяқталған соң жойылады * Басқа функциялар оны тани алмайды * Әр функция өз локальды айнымалыларын жеке сақтайды ### Мысал: ```python id="local_var_full" def calculate(): y = 50 # локальды айнымалы print("Функция ішінде:", y) calculate() # print(y) # қате: y сыртта жоқ ``` Мұнда `y` тек `calculate()` функциясының ішінде ғана бар. --- ## 🔴 Глобальды мен локальды айнымалылардың айырмашылығы | Критерий | Глобальды айнымалы | Локальды айнымалы | | --------------------- | ------------------ | ------------------- | | Қай жерде жарияланады | Функциядан тыс | Функция ішінде | | Қолжетімділігі | Барлық жерде | Тек өз функциясында | | Өмір сүру уақыты | Программа бойы | Функция аяқталғанша | | Қолданылуы | Ортақ деректер | Жеке есептеулер | --- ## ⚠️ Глобальды айнымалыны өзгерту Функция ішінде глобальды айнымалыны өзгерту үшін арнайы `global` кілт сөзі қолданылады: ```python id="global_modify" x = 10 def change(): global x x = 200 change() print(x) ``` Нәтиже: ``` 200 ``` --- ## 🧠 Маңызды түсінік: * Локальды айнымалылар — қауіпсіз және тәуелсіз * Глобальды айнымалылар — ортақ, бірақ қате өзгерістерге әкелуі мүмкін * Жақсы бағдарламада көбіне локальды айнымалылар көбірек қолданылады --- ## 📌 Қорытынды Локальды және глобальды айнымалылар программаның құрылымын дұрыс ұйымдастыруға көмектеседі. Локальды айнымалылар функция ішінде деректерді оқшау сақтаса, глобальды айнымалылар барлық программаға ортақ ақпаратты сақтайды. Оларды дұрыс қолдану кодтың түсініктілігін, қауіпсіздігін және тиімділігін арттырады.
