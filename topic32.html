📘 Кірістірілген циклдер және олардың қолданылуы

Кірістірілген циклдер (nested loops) — бір циклдің ішінде тағы бір циклдің орналасуы. Яғни, сыртқы цикл әр орындалған сайын ішкі цикл толық орындалады. Бұл құрылым көпөлшемді деректермен жұмыс істеуде, кесте түріндегі ақпараттарды өңдеуде және күрделі қайталанатын есептерді шешуде қолданылады. Негізгі идеясы: сыртқы цикл “қадамды” басқарады, ал ішкі цикл сол қадамның ішінде толық орындалады. Мысалы, сыртқы цикл жолдарды, ал ішкі цикл бағандарды өңдеуі мүмкін. Ең қарапайым мысал — кесте шығару. Егер біз 3x3 өлшемді сандар кестесін шығарсақ, сыртқы цикл 3 рет қайталанады, ал әр қайталауда ішкі цикл тағы 3 рет орындалады. Псевдокод түрінде: Басы i = 1-ден 3-ке дейін  j = 1-ден 3-ке дейін   i және j мәндерін шығару  соңы j циклі соңы i циклі Соңы JavaScript мысалы: ```javascript id="n91kq2" for(let i = 1; i <= 3; i++){ for(let j = 1; j <= 3; j++){ console.log("i =", i, " j =", j); } } ``` Бұл жерде: * сыртқы цикл (`i`) — негізгі қайталау саны * ішкі цикл (`j`) — әр сыртқы қадамдағы ішкі әрекет Кірістірілген циклдердің қолданылуы өте кең: * матрицамен (екіөлшемді массив) жұмыс істеу * кесте, тор, координаталар шығару * сурет салу (символдар арқылы фигура жасау) * комбинацияларды есептеу * сұрыптау алгоритмдері (мысалы, bubble sort) Мысалы, жұлдызшалардан шаршы шығару: ```javascript for(let i = 1; i <= 5; i++){ let row = ""; for(let j = 1; j <= 5; j++){ row += "* "; } console.log(row); } ``` Қорытындылай келе, кірістірілген циклдер — күрделі қайталанатын құрылымдарды басқаруға арналған тиімді құрал және көпөлшемді деректермен жұмыс істеудің негізгі тәсілі болып табылады.
