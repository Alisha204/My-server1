📘 break және continue операторлары

break және continue операторлары — циклдердің жұмысын басқаруға арналған арнайы операторлар. Олар циклдің орындалу тәртібін өзгерту үшін қолданылады. break операторы циклді толық тоқтату үшін қолданылады. Яғни, белгілі бір шарт орындалса, цикл бірден үзіліп, одан кейінгі қайталаулар орындалмайды. break көбіне іздеу есептерінде немесе қажетті мән табылған кезде пайдаланылады. Мысалы, массивтен белгілі бір санды тапқанда циклді тоқтату: let arr = [2, 5, 8, 10, 15]; for(let i = 0; i < arr.length; i++){ if(arr[i] == 10){ console.log("Табылды, индексі:", i); break; } } Бұл жерде 10 саны табылған кезде цикл тоқтайды. continue операторы циклдің сол ағымдағы қадамын өткізіп жіберу үшін қолданылады. Яғни, шарт орындалса, сол итерацияның қалған бөлігі орындалмайды, бірақ цикл әрі қарай жалғасады. Мысалы, тек жұп сандарды шығару: for(let i = 1; i <= 10; i++){ if(i % 2 !== 0){ continue; } console.log(i); } Бұл жерде тақ сандар кездессе, continue оларды өткізіп жібереді, ал жұп сандар ғана экранға шығады. Қорытынды: break → циклді толық тоқтатады continue → тек бір қадамды өткізіп, циклді жалғастырады Олар циклдерді тиімді басқаруға және артық есептеулерді азайтуға көмектеседі.
