📘 Функция арқылы санның факториалын есептеу

**Функция арқылы санның факториалын есептеу** — Data Structures және алгоритмдер теориясында жиі кездесетін классикалық есептердің бірі болып табылады. Бұл есептің негізі — берілген натурал санның факториалын табу, яғни 1-ден бастап сол санға дейінгі барлық бүтін сандардың көбейтіндісін есептеу. Факториал математикалық түрде n! деп белгіленеді және ол көптеген алгоритмдерде, комбинаторикада, ықтималдық теориясында және программалауда кеңінен қолданылады. Факториалдың анықтамасы бойынша 0! = 1 және 1! = 1 болып есептеледі. Ал кез келген n ≥ 2 үшін n! = 1 × 2 × 3 × ... × n формуласы орындалады. Мысалы, 5! = 1 × 2 × 3 × 4 × 5 = 120. Осы қарапайым ережені программалық түрде жүзеге асыру үшін функциялар қолданылады, себебі функция кодты қайта қолдануға және құрылымдауға мүмкіндік береді. Факториалды есептеудің екі негізгі тәсілі бар: цикл арқылы (итерациялық әдіс) және рекурсия арқылы. --- ## 1. Цикл (итерациялық) әдісі арқылы функция Бұл әдісте факториал біртіндеп көбейту арқылы есептеледі. Яғни бастапқы мән 1-ге тең болып, әр қадамда келесі санға көбейтіледі. ```python id="fact_iterative_full" def factorial(n): result = 1 for i in range(1, n + 1): result = result * i return result n = int(input("Санды енгізіңіз: ")) print("Факториал:", factorial(n)) ``` Бұл бағдарламада: * `factorial(n)` — функцияның аты * `result = 1` — бастапқы көбейтінді мәні * `for i in range(1, n+1)` — 1-ден n-ге дейін қайталайды * `result = result * i` — әр санды көбейтеді * Соңында нәтиже қайтарылады --- ## 2. Рекурсия арқылы функция Рекурсия — функцияның өз-өзін шақыруы арқылы есепті шешу тәсілі. Бұл әдіс математикалық анықтамаға өте жақын. ```python id="fact_recursive_full" def factorial(n): if n == 0 or n == 1: return 1 else: return n * factorial(n - 1) n = int(input("Санды енгізіңіз: ")) print("Факториал:", factorial(n)) ``` Бұл бағдарламада: * Егер n = 0 немесе 1 болса, нәтиже 1 қайтарылады (негізгі шарт) * Әйтпесе функция өз-өзін `n-1` мәнімен шақырады * Осылайша есеп кішірейіп отырып соңында базалық жағдайға келеді --- ## 3. Есептеу мысалы Енгізу: 5 Есептеу барысы: * 5 × 4 × 3 × 2 × 1 = 120 Шығару: 120 --- ## Қорытынды Функция арқылы факториал есептеу программалауда функция ұғымын, циклдерді және рекурсия принципін түсінуге көмектеседі. Бұл есеп қарапайым болғанымен, ол күрделі алгоритмдік есептердің негізін құрайды және логикалық ойлауды дамытуда маңызды рөл атқарады.
