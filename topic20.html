📘 Цикл ішінде шартты операторды қолдану

fЦикл ішінде шартты операторды қолдану дегеніміз – қайталану барысында белгілі бір шартты тексеріп, соған байланысты әртүрлі әрекеттер орындау. Яғни цикл қайталанып жатқан кезде оның ішінде if, else сияқты шартты операторлар қолданылады, бұл алгоритмді икемді және ақылды етеді. Цикл тек бірдей әрекетті қайталап қана қоймай, әр айналымда әртүрлі нәтиже беруі мүмкін. Бұл үшін циклдің ішінде шарт тексеріледі. Мысалы, 1-ден 10-ға дейінгі сандардың ішінен тек жұп сандарды шығару керек болса, цикл ішінде if шарты қолданылады. Мысалы: ```id="loopif1" for(int i = 1; i <= 10; i++) { if(i % 2 == 0) { // жұп сан System.out.println(i); } } ``` Бұл жерде цикл 1-ден 10-ға дейін қайталанады, ал if операторы әр санды тексеріп, тек жұп сандарды ғана шығарады. Тағы бір мысал: ```id="loopif2" int i = 1; while(i <= 5) { if(i == 3) { System.out.println("Үш саны табылды"); } else { System.out.println(i); } i++; } ``` Мұнда цикл 5 рет орындалады, бірақ 3 болған кезде бөлек әрекет орындалады. Цикл ішінде шартты операторды қолданудың артықшылығы – бір цикл арқылы күрделі логиканы орындауға болады, яғни фильтрация, таңдау, тексеру сияқты әрекеттерді жүзеге асырады. Бұл тәсіл есептерді тиімді және қысқа түрде шешуге мүмкіндік береді. Қорытындылай келе, цикл ішінде шартты операторды қолдану – алгоритмді күшейтетін тәсіл, ол қайталану барысында деректерді тексеріп, әртүрлі нәтижелер алуға мүмкіндік береді.
