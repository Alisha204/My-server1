📘 Функция (процедура) ұғымы

Функция (процедура) дегеніміз – программадағы белгілі бір міндетті орындау үшін жазылған, қайта-қайта қолдануға болатын командалар жиынтығы. Яғни үлкен программаны кішкентай бөліктерге бөліп, әр бөлікті жеке орындауға мүмкіндік беретін құрылым. Функция мен процедураның негізгі мақсаты – кодты қысқарту, оны түсінікті және реттелген ету. Бір рет жазылған функцияны бірнеше жерде қайта қолдануға болады, сондықтан бірдей кодты қайта-қайта жазудың қажеті жоқ. Функция мен процедураның айырмашылығы бар: функция міндетті түрде бір мән (нәтиже) қайтарады, ал процедура ешқандай мән қайтармайды, тек белгілі бір әрекетті орындайды. Мысалы, екі санның қосындысын есептейтін функция нәтижені қайтарады, ал экранға мәтін шығаратын процедура тек әрекет орындайды. Мысалы (функция): ```id="func1" int kosu(int a, int b) { return a + b; } ``` Мұнда kosu функциясы екі санды қосып, нәтижесін қайтарады. Мысалы (процедура): ```id="proc1" void habar() { System.out.println("Сәлем!"); } ``` Бұл процедура тек мәтін шығарады, ешқандай мән қайтармайды. Функциялар мен процедуралар программаны құрылымдауға көмектеседі, яғни үлкен есепті бөліктерге бөліп, әрқайсысын жеке-жеке орындауға мүмкіндік береді. Сонымен қатар, олар кодты оқуды жеңілдетеді және қателерді азайтады. Қорытындылай келе, функция – нәтиже қайтаратын ішкі программа, ал процедура – нәтиже қайтармайтын, тек әрекет орындайтын ішкі программа, және екеуі де программалаудың маңызды элементтері болып табылады.
